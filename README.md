## 実行環境の必要条件
* python >= 3.10
* pygame>= 2.1

## ゲーム概要
主人公キャラクターを矢印キー操作により移動させてゴールを目指すゲーム。

## ゲームの実装
### 共通基本機能
* 主人公キャラクターの描画と移動

### 担当追加機能
